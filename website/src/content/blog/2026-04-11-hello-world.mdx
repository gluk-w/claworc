Welcome to the Claworc blog. This is our first post on the new site.

We rebuilt the landing page on Astro and wired in a git-based content model so
that every page — and every post — is written as MDX, validated by Zod, and
deployed as static files to Cloudflare Workers. Authoring, review, and rollback
all happen in pull requests.

More to come: feature announcements, build notes, and release changelogs.
